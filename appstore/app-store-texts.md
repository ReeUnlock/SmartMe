### Review Notes

```
Thank you for reviewing SmartMe! Here is additional context about the app's features and permissions:

VOICE COMMANDS (Microphone Permission)
SmartMe includes a voice command feature powered by OpenAI's Whisper speech-to-text API. When the user taps the microphone button, audio is recorded via the browser's MediaRecorder API, sent to our backend server, transcribed by OpenAI Whisper, and parsed into structured actions (e.g., "add event", "log expense", "add shopping item"). The user always sees a confirmation dialog before any action is executed. Audio data is processed in real-time and is NOT stored on our servers or OpenAI's servers beyond the duration of processing.

CAMERA & PHOTO LIBRARY (Camera/Photo Permissions)
The camera and photo library are used exclusively for receipt scanning. Users can photograph a receipt or select one from their gallery. The image is processed client-side (resized, compressed) and sent to our backend where Tesseract OCR (running locally on our server, not a cloud service) extracts the store name, date, and total amount. Receipt images are NOT permanently stored — they are processed and discarded. No other camera or photo features exist in the app.

ACCOUNT DELETION
Users can delete their account and all associated data at any time:
Settings → "Danger Zone" section → "Reset Account" → confirm with password → confirm in dialog.
This permanently deletes all user data across 13 database tables (events, expenses, shopping lists, goals, rewards, etc.) and the user record itself. Data is removed immediately from our servers.

AUTHENTICATION
The app uses email + password authentication with JWT tokens. Registration requires email verification. Users can:
- Register at the sign-up screen
- Log in with email and password
- Reset password via email link
- Delete account in Settings

IN-APP PURCHASES
SmartMe does NOT include any in-app purchases. All features are currently available for free. A future Pro subscription plan is planned but not active — all billing/subscription UI is hidden in the iOS build.

- Sentry: Anonymous crash reporting
- PostHog: Analytics (EU servers, opt-in only via cookie consent)
```
